#  Currency Converter Web App

A responsive and user-friendly **Currency Converter** built using modern web technologies. This application allows users to convert currencies in real-time using live exchange rate APIs.

##  Live Demo

 https://currency-converter-pi-sable.vercel.app/

##  Features

*  Convert between multiple international currencies
*  Real-time exchange rates using API
*  Fast and responsive UI
*  Fully responsive (Mobile + Desktop)
*  Easy-to-use interface
*  Error handling for invalid inputs

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** Exchange Rate API
* **Deployment:** Vercel
* **Version Control:** Git & GitHub


##  How It Works

1. User selects **base currency**
2. Enters the amount
3. Chooses **target currency**
4. App fetches live exchange rate from API
5. Displays converted value instantly
